# Python Data Science Libraries
Common Python libraries every data scientist should have in their toolbox

- Numpy: Data
- Pandas: Dataframe 
- Matplotlib: Graph
- Seaborn: Matplotlib++, awesome-looking plots, great integration with Pandas
